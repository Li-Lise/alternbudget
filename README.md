Application web de gestion budgétaire et de suivi des candidatures en alternance,
conçue pour les étudiants en informatique. Réalisée en vibe coding avec Base44
le 19 février 2026.

💡 Pourquoi ce projet ?
En 2026, plutôt que de présenter 5 projets superficiels, j'ai choisi de me concentrer
sur une seule application et de partager au quotidien mon aventure d'auto-apprentissage
comme un carnet de voyage technique.
Ce projet me permet de :

‹ • › M'entraîner sur l'audit de code généré par vibe coding
‹ • › Proposer une feuille de route d'apprentissage cohérente sur GitHub
‹ • › Comprendre autant le jargon des développeurs Fullstack que celui des opérationnels
‹ • › Explorer les problématiques de qualité de service en Systèmes, Réseaux, Cloud et Cybersécurité
appliquées au secteur FinTech

🚀 Fonctionnalités

‹ • › Gestion budgétaire avec règle 50/30/20 (besoins / loisirs / épargne)
‹ • › Catégories adaptées à la vie étudiante IT (abonnements cloud, formations, transport)
‹ • › Mode Galère — détection automatique des semaines difficiles
‹ • › Simulateur de projection financière sur 6, 12 et 24 mois
‹ • › Carte interactive géolocalisée des candidatures avec code couleur par statut
‹ • › Calcul automatique du coût réel de la recherche (transport, temps, relances)
‹ • › ROI symbolique par candidature
‹ • › 📞 Intégration VoIP 3CX via API REST et Python (en cours)
